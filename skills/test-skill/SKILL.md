# Test Skill

This is a test skill for the anvil workflow system.

## Purpose

Validates that skills are loaded correctly from the package-local directory.

## How It Works

When loaded, this skill instructs the agent to:

1. Acknowledge it has loaded the test skill
2. Confirm the skill was sourced from the anvil package
3. Proceed with the user's request normally

## Verification

If you see this message, skill loading is working correctly.
